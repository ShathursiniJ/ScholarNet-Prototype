# 🎓 ScholarNet — E‑Learning Mobile App Prototype

**ScholarNet** is a Figma mobile prototype that showcases UI/UX design and user flows for a modern e‑learning platform.

🔗 **Figma Prototype:**
[https://www.figma.com/design/RDTGmgiB4jhgUjiPjHQ3uU/ScholarNet---Poster?node-id=40-1576&t=cjWx3iiIKbetXASL-1](https://www.figma.com/design/RDTGmgiB4jhgUjiPjHQ3uU/ScholarNet---Poster?node-id=40-1576&t=cjWx3iiIKbetXASL-1)

---

## 📌 Overview

ScholarNet demonstrates core screens and interactions including course browsing, learning modules, quizzes, progress tracking, and user account features — all optimized for mobile users.

---

## 🎯 Purpose

This prototype is designed to:

* Present UI/UX designs and interactions
* Visualize key user journeys
* Support usability reviews and feedback
* Serve as a reference for developers
* Guide future product planning

---

## 🧩 Scope

The prototype includes:

✔ Onboarding and Authentication
✔ Home & Course Discovery
✔ Course Detail & Enrollment
✔ Learning Modules & Videos
✔ Quizzes & Assessments
✔ Progress Tracking
✔ Profile & Settings
